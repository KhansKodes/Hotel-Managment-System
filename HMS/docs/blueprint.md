# **App Name**: BudgetFlow

## Core Features:

- Monthly Budget Cap: Set and edit a monthly budget cap. This serves as the basis for all financial calculations.
- Expense Logging: Log expenses with details like date, amount, category, and description.  Option to select or add a new category.
- Real-time Calculations: Automatically calculate total expenses, remaining budget, daily allowance, average daily spend, and projected month-end spend.
- Data Visualization: Visualize financial data using charts like category pie chart, 30-day spend line, and daily spend bar chart.
- Data Persistence: Data will be stored to Firebase. Ensure data is saved correctly after a page refresh or browser close.
- Import/Export: Import and export financial data using JSON and CSV formats to secure data and/or enable transfers across devices.

## Style Guidelines:

- Primary color: Teal (#008080) to reflect balance and growth, aligning with financial themes.
- Background color: Light cyan (#E0FFFF) for a clean and unobtrusive backdrop that complements the primary color without overwhelming the user.
- Accent color: Lime green (#32CD32) to highlight positive budget states or important actions, providing a vibrant contrast against the teal.
- Body: 'PT Sans', a humanist sans-serif which will be applied for all text.
- Use minimalistic, clear icons to represent expense categories and actions.
- Implement a clean, mobile-first layout with intuitive navigation. Prioritize key information like remaining budget prominently.
- Use subtle transitions and animations to provide feedback on user actions and guide attention without being distracting.
